1. Настраиваем и запускаем сервер "kaillerasrv.exe"
1. Запускаем эмулятор (gens.exe)
2. Выбираем Файл>Сетевая
3. В новом окне находим вкладку Recent, вводим "Username"
   находим или вводим наш сервер с помощью кнопки "Enter IP..."
   для локальной машины это может быть 127.0.0.1 или 127.0.0.1:27888 (с указанием порта),
   для удаленной машины IP адрес и порт соответственно
4. Нажимаем Connect - откроется чат
5. Чтобы подключиться к создаваемой игре, надо выбрать игру внизу из списка и нажать Join
   Чтобы создать игру, надо нажать Create New Game и выбрать игру из списка
6. После присоединения всех участников, тот кто создал игру кажимает Start Game

Примечание:
Путь к папке с играми должен быть указан в конфиге Gens.cfg

	[General]
	Rom path=.\_Games\
